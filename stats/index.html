Lathe.stats.confints — Function

Returns the confidence intervals of given data.

array = [5,10,15]

confidence = .98

low, high = Lathe.stats.confints(array,confidence)

PARAMETERS

confidence: Confidence is a float percentage representing the level of confidence required in your test. The confidence metric is used exclusively for calculating the interval.

Ranks indices in an array based on quantitative weight (count of the numbers) and returns a new array of the ranks of each column. This function is made primarily for the Wilcox Rank-Sum test.
