Lathe.preprocess.SortSplit — Function

SortSplit sorts the data from least to greatest, and then splits it, ideal for quartile calculations.

array = [5,10,15]

top25, lower75 = Lathe.preprocess.SortSplit(array,at = 0.75,rev = false)

PARAMETERS:

at:: Percentage value used to determine a point to split the data.

rev:: Reverse, false by default, determines whether to sort least to greatest, or greatest to least.

Lathe.preprocess.UniformSplit — Function

Uniform Split does the exact same thing as ArraySplit(), but observations are returned split, but unsorted and unshuffled.

array = [5,10,15]

test, train = Lathe.preprocess.UniformSplit(array,at = 0.75)

PARAMETERS:

at:: Percentage value used to determine a point to split the data.
